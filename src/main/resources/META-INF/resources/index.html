API Information

REST API Endpoints
The parser module exposes several REST endpoints for testing and integration.

Available Endpoints

GET /modules/parser/api/parser/service/config

Get the JSON Schema for parser configuration

POST /modules/parser/api/parser/service/parse-file

Parse an uploaded file with multipart/form-data

POST /modules/parser/api/parser/service/parse-json

Parse with JSON payload including configuration

GET /modules/parser/api/parser/service/info

Get module information and metadata

GET /modules/parser/q/openapi

OpenAPI 3.1 specification for all endpoints

gRPC API

The parser also provides a gRPC API implementing the PipeStepProcessor service. This is the primary interface for production use within the Pipestream platform.
